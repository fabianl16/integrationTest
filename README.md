## Installation

1. Clone the repository:
```bash
git clone https://github.com/fabianl16/actualSalesTest
cd actualSalesTest
```
2.Install PHP dependencies:
```
composer install
```
3.Install and compile frontend dependencies (only if applies):
```
npm install
npm run dev
```
4.Configure environment:
```
cp .env.example .env
php artisan key:generate
```
6.Run migrations:
```
php artisan migrate
```
